# Saved User Context

This file captures the current project intent so the user does not have to
repeat the same plan in future work.

- The user wants the desktop pet chatbot to preserve context across chats and
  restarts.
- The repeated pain point is having to restate the plan because the chatbot does
  not remember prior project context.
- The desired mechanism should be efficient, evolve over time, and learn durable
  user preferences, needs, work style, and project plans.
- The project is a Windows desktop pet built with PyQt6, SQLite, Ollama local
  chat/cloud fallback, todos, work tracking, proactive nudges, and privacy-safe
  coarse app buckets.
- The preferred privacy stance is summarized local memory rather than raw chat
  transcript storage.
